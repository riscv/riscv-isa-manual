== "Ss" Supervisor Extensions

NOTE: This chapter is currently being restructured.
Its contents are normative, but the presentation might appear disjoint.

[[ssccptr]]
=== Ssccptr Extension for Main Memory Page-Table Reads, Version 1.0

If the Ssccptr extension is implemented, then main memory regions with both
the cacheability and coherence PMAs must support hardware page-table reads.

[[sstvecd]]
=== Sstvecd Extension for Direct Trap Vectoring, Version 1.0

If the Sstvecd extension is implemented, then `stvec.MODE` must be capable of
holding the value 0 (Direct).
Furthermore, when `stvec.MODE=Direct`, `stvec.BASE` must be capable of holding
any valid four-byte-aligned address.
