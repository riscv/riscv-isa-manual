[[ssstrict]]
=== Ssstrict Extension for Extension Conformance, Version 1.0

[[norm:ssstrict_no_nonconforming_extensions]]
If the Ssstrict extension is implemented, then no non-conforming extensions
are present.

[[norm:ssstrict_unimplemented_opcodes_csr_exception]]
Furthermore, attempts to execute unimplemented opcodes or access
unimplemented CSRs in the standard or reserved encoding spaces raises an
illegal instruction exception that results in a contained trap to the
supervisor-mode trap handler.
