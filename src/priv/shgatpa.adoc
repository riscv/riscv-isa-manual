[[shgatpa]]
=== Shgatpa Extension for Translation Mode Support, Version 1.0

[[norm:shgatpa_satp_hgatp_mode_support]]
If the Shgatpa extension is implemented, then for each supported virtual
memory scheme Sv__NN__ supported in `satp`, the corresponding hgatp Sv__NN__x4
mode must be supported.

[[norm:shgatpa_hgatp_bare_mode]]
Furthermore, the `hgatp` mode Bare must also be supported.
