[[ext:zve64f]]
=== ext:zve64f[] 64-bit Integer and Single-Precision Floating-Point Vector Extension

This section describes ext:zve64f[], the vector extension for
single-precision floating-point and 64-bit integer and fixed-point types.
It is compatible with all currently defined base ISAs.

[#norm:Zve64f_dependent_Zve32f_Zve64x]#ext:zve64f[] depends on the ext:zve64x[] and ext:zve32f[] extensions.#
