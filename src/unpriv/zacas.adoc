[[ext:zacas]]
=== ext:zacas[] Extension for Atomic Compare-and-Swap (CAS) Instructions

Compare-and-swap (CAS) provides an easy and typically faster way to perform
thread synchronization operations when supported as a hardware instruction. CAS
is typically used by lock-free and wait-free algorithms. The ext:zacas[]
extension is shorthand for the following set of extensions:

[%header,cols="^1,4"]
|===
|Included Extension
|Description

| extlink:zacasw[] | Word (32-bit) CAS instructions.
| extlink:zacasd[] | Doubleword (64-bit) CAS instructions.
| extlink:zacasq[] | Quadword (128-bit) CAS instructions (RV64 only).
|===

[#norm:Zacas_impl_all]#A core which implements ext:zacas[] must implement all of
the above extensions.#
