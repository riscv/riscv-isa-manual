[[ext:zve32x]]
=== ext:zve32x[] 32-bit Integer Vector Extension

This section describes ext:zve32x[], the vector extension for 8-, 16-,
and 32-bit integer and fixed-point types.
[#norm:Zve_XLEN]#ext:zve32x[] is compatible with all currently defined base ISAs.#

The ext:zve32x[] extension requires that VLEN be at least ELEN.

[#norm:Zve_precise_traps]#The ext:zve32x[] extension has precise traps.#

NOTE: There is currently no standard support for handling imprecise
traps, so standard extensions have to provide precise traps.

[#norm:Zve_eew]#ext:zve32x[] provides support for EEW of 8, 16, and 32.#

ext:zve32x[] supports the vector configuration instructions
(<<sec-vector-config>>).

ext:zve32x[] supports all vector load and store instructions
(<<sec-vector-memory>>).

ext:zve32x[] supports all vector integer instructions
(<<sec-vector-integer>>).

ext:zve32x[] supports all vector fixed-point arithmetic
instructions (<<sec-vector-fixed-point>>).

ext:zve32x[] supports all vector integer single-width and
widening reduction operations (<<sec-vector-integer-reduce>>,
<<sec-vector-integer-reduce-widen>>).

ext:zve32x[] supports all vector mask instructions
(<<sec-vector-mask>>).

[#norm:Zve32x_Zve64x_nsupport_freg]#ext:zve32x[] supports all vector integer permutation instructions
(<<sec-vector-integer-permute>>).#

[#norm:Zve32x_dependent_Zicsr]#The ext:zve32x[] extension depends on the Zicsr extension.#
