== RISC-V Memory Models

NOTE: This chapter is currently being restructured.
Its contents are normative, but the presentation might appear disjoint.

This chapter describes the two RISC-V memory consistency models:
<<memorymodel,RVWMO>>, the base weakly ordered model, and <<ext:ztso,RVTSO>>,
a more strongly ordered model enabled via the ext:ztso[] standard extension.

Additional explanatory material for both models can be found in
<<app:mm-explanatory>>.
