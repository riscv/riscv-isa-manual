[[zama16b]]
=== Zama16b Extension for 16-byte Misaligned Atomicity, Version 1.0

If the Zama16b extension is implemented, then the
<<sec:misaligned-atomicity-granule,misaligned atomicity granule>>
in main memory regions with both the cacheability and coherence PMAs is 16
bytes.
Misaligned loads, stores, and AMOs to main memory regions that do not cross
a naturally aligned 16-byte boundary are atomic.
