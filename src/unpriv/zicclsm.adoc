[[ext:zicclsm]]
=== ext:zicclsm[] Extension for Main Memory Misaligned Accesses

[[norm:zicclsm_misaligned_support]]
If the ext:zicclsm[] extension is implemented, then misaligned loads and stores
to main memory regions with both the coherence and cacheability PMAs must be
supported.

NOTE: This definition includes vector memory accesses.
It does not include any instructions in the various ext:Za*[] extensions.

NOTE: Even though mandated, misaligned loads and stores might execute
extremely slowly.
Standard software distributions should assume their existence only for
correctness, not for performance.
