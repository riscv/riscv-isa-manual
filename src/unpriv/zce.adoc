[[ext:zce]]
=== ext:zce[] Standard Extension for Enhanced Instruction Compression

This section describes the ext:zce[] extension, which incorporates the
compressed instruction-set extensions designed for microcontrollers.
Unlike the ext:c[] extension, the ext:zce[] extension includes extensions
that trade performance for code density.

The ext:zce[] extension depends upon the extlink:zca[], extlink:zcb[],
extlink:zcmp[], and extlink:zcmt[] extensions.

If XLEN=32 and the F extension is present, the ext:zce[] extension
additionally depends upon the extlink:zcf[] extension.
