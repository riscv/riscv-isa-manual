[[ext:zvfhmin]]
=== ext:zvfhmin[] Vector Extension for Minimal Half-Precision Floating-Point

The ext:zvfhmin[] extension provides minimal support for vectors of IEEE 754-2008
binary16 values, adding conversions to and from binary32.
When the ext:zvfhmin[] extension is implemented, the insn:vfwcvt.f.f.v[] and
insn:vfncvt.f.f.w[] instructions become defined when SEW=16.
The EEW=16 floating-point operands of these instructions use the binary16
format.

[#norm:Zvfhmin_dependent_Zve32f]#The ext:zvfhmin[] extension depends on the extlink:zve32f[] extension.#
