[[ext:zve32f]]
=== ext:zve32f[] Single-Precision Floating-Point Vector Extension

This section describes ext:zve32f[], the vector extension for
single-precision floating-point.
It is compatible with all currently defined base ISAs.

[#norm:Zve32f_Zve64f_dependent_F]#The ext:zve32f[] extension depends on the ext:zve32x[] and ext:f[] extensions.#

ext:zve32f[] implements all vector floating-point instructions
(<<sec-vector-float>>) for floating-point operands with EEW=32.
Vector single-width floating-point reductions
(<<sec-vector-float-reduce>>) for EEW=32 are supported.
