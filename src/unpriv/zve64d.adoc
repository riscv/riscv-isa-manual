[[ext:zve64d]]
=== ext:zve64d[] Double-Precision Floating-Point Vector Extension

This section describes ext:zve64d[], the vector extension for
double-precision floating-point.
It is compatible with all currently defined base ISAs.

[#norm:Zve64d_dependent_Zve64f]#The ext:zve64d[] extension depends on the ext:zve64f[] and ext:d[] extensions.#

ext:zve64d[] implements all vector floating-point instructions
(<<sec-vector-float>>) for floating-point operands with EEW=32
or EEW=64 (including widening instructions and conversions between
FP32 and FP64).
Vector single-width floating-point reductions
(<<sec-vector-float-reduce>>) for EEW=32 and EEW=64 are supported as
well as widening reductions from FP32 to FP64.
