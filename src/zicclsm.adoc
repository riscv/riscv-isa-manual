[[zicclsm]]
=== Zicclsm Extension for Main-Memory Misaligned Accesses, Version 1.0

If the Zicclsm extension is implemented, then misaligned loads and stores to
main memory regions with both the cacheability and coherence PMAs must be
supported.

NOTE: This definition includes vector memory accesses.
It does not include AMOs, LR, or SC, as the Zaamo and Zalrsc extensions
require those instructions raise exceptions upon misalignment.

NOTE: Even though mandated, misaligned loads and stores might execute
extremely slowly.
Standard software distributions should assume their existence only for
correctness, not for performance.
