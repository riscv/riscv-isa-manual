== Preface

This document describes the RISC-V architecture profiles.
It contains the following profiles,
all of which have been ratified:

[%autowidth,float="center",align="center",options="header"]
|===
|Profile
|*RVI20U32*
|*RVI20U64*
|*RVA20U64*
|*RVA20S64*
|*RVA22U64*
|*RVA22S64*
|*RVA23U64*
|*RVA23S64*
|*RVB23U64*
|*RVB23S64*
|===

[.big]*_Changes made since ratification of RVA22 profiles_*

- Clarified that `Zihpm` was optional in RVA20U64 and became mandatory in
  RVA22U64

[.big]*_Changes made since public review of RVA22 profiles_*

- Clarified that profile name can be used as ISA base string
- Renamed `Ssptead` to `Svade`
- Fixed `Ssu64xl` to make supporting UXL=64 mandatory
- Added section listing new extension names in profiles document
- Added new extension name `Sscounterenw`
- Removed outdated text on `Zicntr`/`Zihpm` ratification plan
